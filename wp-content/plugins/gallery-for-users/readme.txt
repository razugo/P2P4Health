== Description ==
Allow your users to display their images and videos with this flexible user gallery plugin.

**Main features of the plugin**

* Users can upload either images or link to their Youtube videos 
* Search for images/videos using various filters
* The user can see list of his own images/videos and can delete them
* Images and videos can be shared on social media
* Admin can adjust colors for buttons and text as well as select font for design flexibility
* Admin can create categories, which images/videos can be added to

**Now available in the pro version!**

* Add a widget, displaying the newest additions to the gallery
* Admin can control which user groups are allowed to add to the gallery
* Admin can view (and delete) any image or video from backend
* Admin can add images or videos from backend

**[Read more about the pro version here](https://awesometogi.com/gallery-for-users-plugin-for-wordpress/)**

== Installation ==
1. Log into your Wordpress admin dashboard and search for ‘Gallery for users’ in ‘Plugins’ - ‘Add new’

2. Activate the plugin through the ‘Plugins’ screen in Wordpress 

3. That’s it. The gallery settings have been added as a new menu item in your dashboard, so find the menu item named \'User Gallery\'
